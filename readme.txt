w2pe Measurement Widget is especially designed to make your units conversion job a whole lot easier. Here you'll find instant conversions for thousands of various units and measurements.

== Description ==

w2pe Measurement Widget is a simple measurement converter. You can use both in page or widget. There is a possibility to add custom units in admin area and edit or delete units.
Measurements and units included as presets: 
* Area - foot, inch, meter, mile, kilometer
* Currency - Bangladeshi TK, British pound, Euro, Indian rupee, Japanese Yen, US dollar
* Length - kilometer, meter,  mile, yard
* Mass - gram, kilogram, pound, ton
* Temperature - Celsius, Fahrenheit, Kelvin

= How to use? =

1. If you want to use in widget, use "w2pe Measurement Widget"
2. If you want to use in a page or post, use [w2pe_measurement]

== Installation ==

1. Upload the entire `w2pe-measurement-widget` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'w2pe-beaverslider' menu in your WordPress admin panel.

For basic usage, you can also have a look at the [plugin homepage](http://www.wppluginexpert.com/) plugin section.
